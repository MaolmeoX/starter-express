## Launch

### Step 1

```
npm ci
```

### Step 2

```
cp .env.example .env
```

_set JWT SECRET_

### Step 3

Get your own api key from [the movie db.org](https://developers.themoviedb.org/3/getting-started) and set it in .env

### Step 4

```
docker-compose up -d
```

### Step 5

```
npm run start
```
